# Backlog Projects

Planned but not yet started. Ideas waiting for their turn.

## Guidelines

- Keep projects here until you're ready to commit
- Review during weekly planning
- Move to `05_Projects/` root when starting
- Delete if no longer relevant

## When to Start

Consider moving to active when:
- You have capacity (not at 2-3 active limit)
- It aligns with current goals
- You can commit to making progress
