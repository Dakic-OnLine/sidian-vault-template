# Setup Guide

Getting your Sidian Vault running in Obsidian.

---

## Step 1: Create Your Vault

Use the **"Use this template"** button on GitHub to create your own repository, then clone it locally.

---

## Step 2: Open in Obsidian

1. Open Obsidian
2. Click **"Open folder as vault"**
3. Select your cloned folder
4. Done

---

## Step 3: Configure Templates

1. Go to **Settings** → **Core Plugins** → Enable **Templates**
2. Go to **Settings** → **Templates**
3. Set **Template folder location**: `99_Assets/templates`

---

## Step 4: Install Obsidian Git (Recommended)

1. Go to **Settings** → **Community Plugins** → Turn off Restricted mode
2. Click **Browse** → Search "Obsidian Git" → Install
3. Configure auto-commit interval (e.g., 30 minutes)

---

## Step 6: Start Using

1. Open `Home.md` - your dashboard
2. Create a diary entry in `02_Diary/YYYY-MM-DD.md`
3. Capture thoughts in `00_Inbox/Inbox.md`
4. Read `Guide.md` for the full system

---

**That's it.** Read `Guide.md` when you're ready to go deeper.
